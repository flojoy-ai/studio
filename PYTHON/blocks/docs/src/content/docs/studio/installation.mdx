There are **2 steps** to install Flojoy Studio:

## 1. Install Python ~3.11

If you haven't yet installed Python ~3.11 on your machine, please install it
first. You can install Python v3.11 from official website [here](https://www.python.org/downloads/release/python-3116)

## 2. Install Flojoy Studio

Once you have Python ~3.11 installed on your machine, you are ready to install
Flojoy Studio. Simply choose the right installer for your machine and follow the
instructions.
